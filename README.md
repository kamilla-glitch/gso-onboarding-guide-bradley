# GSO Onboarding Guide — Onboarding Specialist

> **Internal use only.** This guide is for new Onboarding Specialists joining the APAC Global Seller Onboarding (GSO) team at Square.

---

## What is this?

A self-contained, single-file interactive onboarding guide built in HTML/CSS/JS. It covers the full 5-week onboarding journey for a new GSO Onboarding Specialist, including:

- **Week 1** — Welcome, team orientation, tooling setup, Slack channels, recurring meetings
- **Week 2** — Cross-functional partners, BPO, 3P relationships, SAMPs
- **Week 3** — Bootcamp agenda and preparation
- **Week 4/5** — Product verticals (Restaurants, Retail, Services, Ecommerce, Staff, Engagement)
- **Week 6–10** — Premium DSRs, escalation workflow

## How to use

1. Open `index.html` in any modern browser (Chrome recommended)
2. No build step, no dependencies, no server required — it's a single file
3. Progress is saved automatically in `localStorage`
